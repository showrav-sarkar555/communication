# Communication Engineering Study Notes

A comprehensive web-based study guide for Communication Engineering, covering 10 essential lectures on data communication and networking.

## üìö Topics Covered

1. **Lecture 01** - Introduction to Data Communication
   - Network Types, Topologies, Protocols, Data Flow

2. **Lecture 02** - Network Models  
   - TCP/IP Model, OSI Model, IP Addressing

3. **Lecture 03** - Physical Layer
   - Signals, Bandwidth, Nyquist Theorem, Shannon Capacity

4. **Lecture 04** - Digital Transmission
   - Line Coding (NRZ, Manchester, AMI), Scrambling

5. **Lecture 05** - Analog Transmission
   - ASK, FSK, PSK, QAM Modulation Techniques

6. **Lecture 06** - Bandwidth Utilization
   - FDM, TDM, Multiplexing, CDMA

7. **Lecture 07** - Transmission Media
   - Twisted Pair, Coaxial Cable, Fiber Optic

8. **Lecture 08** - Switching
   - Circuit Switching, Datagram, Virtual Circuit

9. **Lecture 09** - Data Link Layer
   - Framing, Flow Control, ARP Protocol

10. **Lecture 10** - Error Detection and Correction
    - Parity Check, CRC, Checksum, Hamming Codes

```bash
open index.html   # macOS
xdg-open index.html  # Linux
```
